Projet Wordle Tahiti en association avec l'association Tahiti DevOps et le Service de la Traduction et de l'Interprétariat.

## Stack technologique

- Next.js
- Tailwind CSS
- next-translate
- heroicons
- date-fns
- use-longpress

## Développement

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).
